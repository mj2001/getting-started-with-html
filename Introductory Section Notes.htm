<b>Introductory Section Notes:</b><br>

<p>The <b>WWW consists of</b> me (my computer and browser), the Internet, and one or more servers that host files.  The messages that pass between these components use the <b>hypertext transfer protocol (aka HTTP) </b>.  Web pages are written in <b>HTML, hypertext markup language</b>, which enables browsers and servers to communicate with each other regarding web page layout, content, files, etc.

<p>HTML is composed of <b>tags</b>.  A string/structure consisting of an opening tag, text and a closing tag is called <b>an element</b>.  For example, <br>

<b> This <b>bold element</b> makes this statement bold.</b>  <br>

<em>This <b>emphasis element</b> italicizes this statement.</em>

<p>Note that the closing tags above are delimiters:  They tell the browser where to stop applying the instruction that the tag communicates.  If we omit the closing tag, the instruction will be applied to everything that follows.

<p>Some HTML  tags have attributes.  For example, the <b>anchor tag</b>: 

<p><a href=”www.udacity.com”>This is a link to the Udacity homepage.</a>

<p>Some HTML tags are <b>void tags</b>, meaning that they consist only of opening tags.  For example,<br>

<img src=”imagename.gif“  alt="this image tag links to an image, but it is not followed by a closing tag. ">

<br>and<br>

this <b>break tag</b> introduces a line break into the text:  It skips to the next line in the whitespace that comprises the whole webpage.

<p>Another way of introducing a line break into text is to use the <b>paragraph tag</b>:

<p>This paragraph tag makes a paragraph out of the text that is between its opening and closing tags.</p>  (This is not a void tag.)

<p>Note that the <b>break tag</b> is an <b>inline element</b>, meaning that it ends the line where it occurs.  On the other hand, the <b>paragraph tag</b> is a <b>block element</b> that creates a block of text that is both preceded and followed by a blank line. 

<p><b>Container elements  span and div</b> are further examples of inline and block elements, respectively.  These elements do nothing but contain content and control its layout.  For example,

<p><span class=”foo”>Text</span>    (inline) and

<p><div class=”bar”>Text</div>    (block).

<p>Important things to take away from this lesson:

<p>1 - What a Web Page is:<br>

A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.

<p>2 - Basic HTML Vocabulary:<br>

You will be using HTML in the next few lessons, so it will be helpful if you're comfortable with the jargon.

<p><b>Tag</b>: An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags (called "void" tags) do not require a closing tag (like the break tag).

<p><b>Element</b>: An HTML element refers to everything within a set of opening and closing tags.

<p><b>Attribute</b>: This is a property of an HTML element.
